
Supplementary material for 
Jacobson et al., Quantifying the response of Blainville's beaked whales to US Naval sonar exercises in Hawaii
Updated 2022-01-16

This repository contains code and data to replicate the analyses presented in Jacobson et al. The file 01_MultiGAMAnalysis.R replicates the multi-GAM analysis presented in the main manuscript, while 02_S3SingleGAM.R replicates the single GAM presented in S3. 

These analyses were done in R v. 4.1.0

Required R packages are:
dyplyr
ggplot2
mgcv
scam
tidyr
